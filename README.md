## Homelab — defense in depth

```text
Proxmox VE (segmenté en VLANs : DMZ · Services · Sécurité · Management)
   ├── OPNsense ............ firewall / routeur (least privilege, block par défaut)
   ├── Suricata IDS ........ inspection réseau
   ├── Wazuh SIEM .......... détection, corrélation, MITRE ATT&CK
   ├── Traefik ............. reverse proxy + TLS Let's Encrypt
   └── Cloudflare Zero Trust  accès distant (WARP + Tunnel)
```

Un terrain de jeu pour pratiquer la détection, le hardening et la réponse à incident — et pour faire mûrir les outils ci-dessus en conditions réelles.
